# IOT-Based-Obstacle-Detector
A real-time obstacle detector using IOT detects the object/obstacle and intimates us(by the LEDs)
--------------------------------------------------------------------------------------------------
#### Components :
  - Arduino UNO
  - Ultrasonic Sensor SR-04
  - Jumper Wires
  - LEDs
